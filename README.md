# Jefnix - Flask Movie Recommender

A basic practice project for learning Flask fundamentals: routing, dynamic URLs and Jinja templating.

## Focus
This project's main focus is backend logic that is Flask routes, passing data to templates and dynamic URL parameters. 
Styling was improved with the help of AI assistance, as HTML/CSS is a separate skill I intend to strengthen going forward.

## Features
- Home page with featured movies
- Full movies listing page
- Individual movie detail pages via dynamic routes

## Tech
- Flask
- Jinja2 (templating)
- HTML/CSS

## Run Locally

```bash
pip install flask
python app.py
```

Then open **http://127.0.0.1:5000** in your browser.
